|------|-------|------|
| Tools | ✓ | ✓ |

## Command Line CLI Supported

|------|-------|--------|-----|-------|-------|--------|
| Tools | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |

## Clouds Supported

|------|-------|--------|-----|
| Tools | ✓ | ✓ | ✓ |

## Usage

```shell
docker compose run --rm tools bash -c '
  python -V
  pip -V
  cookiecutter --version
  aws --version
  az --version
  gcloud --version
  dbt --version
  kubectl version
  terraform -version
  terragrunt -version
  pwd'
```
